Paisa Bazar Credit Score Analysis Dashboard

Project Overview
This project analyzes customer credit behavior using financial and demographic data. The analysis was performed in Python using Google Colab, and the insights were visualized in an interactive Power BI dashboard.

The dashboard helps identify trends in income, debt, utilization ratio, and customer credit scores to support better lending decisions.
